The Project

Hazardous Asteroid Detection System, a.k.a H.A.D.S is a project we developed to identify hazardous asteroids based on the size and velocity of the object.


Our Mission

Space, a void of uncertainties and unanswered questions. Where asteroids roam free around the earth. Our team was inspired to work on an AI that can detect dangerous asteroids in order to protect the earth. With curiosity leading our team, we were able to create H. A. D. S.


The Journey The Timeline of the Project

Week 1:
  • Learned Python basics and packages such as NumPy and Pandas
  • Researched and found dataset on Kaggle
  • Started to remove and clean unnecessary data from dataset
  • Used Plotly to create graphs and charts like scatter plots, pie charts, and more.
Week 2:
  • Finished cleaning dataset
  • Researched supervised classification models for the project.
  • Trained & fitted the classification models.
  • Checked models for accuracy and picked high perfomance models
  • Learned about random over-sampling and fixed the imbalance problem.
Week 3:
  • Learned HTML and CSS
